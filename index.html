Replacing Alcatel with Ubiquiti

Technologies: Ubiquiti UniFi, VLANs, PoE, Cloud Management

Led network infrastructure migration at a campus-style facility, replacing legacy Alcatel switches with modern Ubiquiti UniFi infrastructure. Executed comprehensive site assessments, hardware procurement, and coordinated with cross-functional teams to minimize downtime. Configured VLAN segmentation, PoE power budgets, and implemented centralized cloud-based management through UniFi Controller. The upgrade delivered improved reliability, enhanced monitoring capabilities, and scalable architecture supporting 500+ users with 99.9% uptime.

Building Customer Failover Infrastructure

Technologies: Datto, VPN, Data Replication, Rack Infrastructure

Architected and deployed a customer failover datacenter solution within a Tier 3 facility, utilizing dedicated rack space to create comprehensive redundancy for MSP clients. Executed site planning, hardware procurement, and installation of servers, networking equipment, and redundant power systems. Configured automated backup processes using Datto, implemented real-time data replication, and established secure VPN tunnels for remote access to replicated environments. The solution enables clients to achieve RTO < 1 hour and RPO < 15 minutes, ensuring business continuity during critical disruptions.

Home Lab Infrastructure

Technologies: Proxmox, OPNsense, Windows Server 2019, Cisco CML, Active Directory

Designed and implemented a production-grade home lab featuring a Proxmox high-availability cluster, OPNsense firewall with advanced security policies, Windows Server 2019 Active Directory domain, and Cisco CML for virtualized network topologies. The infrastructure includes multi-VLAN segmentation, inter-VLAN routing, OSPF/EIGRP protocol testing, and container orchestration. This environment enables hands-on experimentation with enterprise technologies, certification preparation, and safe testing of configurations before production deployment.
